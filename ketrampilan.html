KETERAMPILAN

Programming Languages

HTML

Memiliki pemahaman yang kuat tentang HTML, bahasa markup dasar untuk menyusun konten web. Mampu membuat dokumen HTML yang semantik, menyusun konten dengan tag dan elemen, dan memahami hierarki halaman web.

CSS

Mahir dalam menggunakan CSS untuk menata halaman web. Dapat mengontrol tata letak, warna, font, dan tampilan visual elemen web. Memahami konsep seperti box model, selector, dan media query untuk desain responsif.

JavaScript

Memiliki pengetahuan tentang JavaScript, bahasa pemrograman dinamis yang digunakan untuk menambahkan interaktivitas ke halaman web. Memahami konsep pemrograman dasar, manipulasi DOM, dan penanganan peristiwa.

Frameworks & Libraries

jQuery

You have experience with jQuery, a fast and concise JavaScript library that simplifies DOM manipulation, event handling, and AJAX interactions.

Bootstrap

Memahami Bootstrap, framework CSS populer untuk membuat situs web yang responsif dan mobile-first. Mampu menggunakan sistem grid, komponen, dan utilitasnya untuk membangun antarmuka web dengan cepat.

Sass

Familiar dengan Sass, preprocessor CSS yang memperluas CSS dengan fitur-fitur seperti variabel, mixin, dan nesting, sehingga membuatnya lebih mudah dikelola dan efisien.

Tools

GitHub

Familiar dengan Git dan GitHub, menggunakan kontrol versi untuk mengelola kode, berkolaborasi dengan orang lain, dan melacak perubahan pada proyek.

Soft Skills

Critical Thinking

Memiliki keterampilan berpikir kritis yang memungkinkan untuk menganalisis informasi secara objektif, mengidentifikasi masalah, dan menyusun solusi yang efektif. Mampu mengevaluasi argumen, membuat deduksi logis, dan mendekati masalah secara sistematis.

Hardware Repair and Maintenance

Computer Repair

Memiliki pengalaman praktis dalam mendiagnosis dan memperbaiki masalah perangkat keras komputer. Ini termasuk mengidentifikasi komponen yang rusak, mengganti suku cadang, dan memastikan komputer berfungsi dengan benar.

Windows Installation

Mampu menginstal dan mengkonfigurasi sistem operasi Windows. Dapat menangani proses instalasi, mengatur driver, dan mengoptimalkan sistem operasi untuk kebutuhan pengguna.

Computer Parts Replacement

Dapat mengidentifikasi dan mengganti komponen komputer yang rusak atau usang seperti RAM, hard drive, dan motherboard. Memahami kompatibilitas antara bagian-bagian yang berbeda dan dapat melakukan penggantian dengan aman.

LAN Installation

Dapat mengatur Jaringan Area Lokal (LAN) untuk rumah atau bisnis kecil. Ini termasuk menjalankan kabel, memasang konektor, dan mengkonfigurasi perangkat jaringan.
