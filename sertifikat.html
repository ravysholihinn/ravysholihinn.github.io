Sertifikat

Sertifikat Berpikir Kritis

Sertifikat ini mengakui kemampuan untuk meningkatkan kualitas hidup melalui penerapan keterampilan berpikir kritis.

  • Analisis Informasi: Memproses dan menilai informasi dari berbagai sumber.
  • Penilaian Argumen: Evaluasi argumen, bukti, dan deteksi bias.
  • Pemecahan Masalah: Mengidentifikasi masalah dan solusi kreatif.
  • Pengambilan Keputusan: Keputusan berbasis analisis kritis.
  • Refleksi Pribadi: Evaluasi keputusan untuk pembelajaran berkelanjutan.

Sertifikat ini menunjukkan kemampuan dalam menangani tantangan hidup secara efektif.

Sertifikat HTML dan CSS
  • HTML: Elemen, tag, atribut, struktur semantik.
  • CSS: Selektor, layout, Flexbox, Grid, media queries.
  • Desain Responsif: Penyesuaian tampilan untuk semua perangkat.
  • Standar Web: Penulisan HTML/CSS yang baik dan sesuai praktik terbaik.

Pelatihan ini membekali keterampilan dasar pengembangan web yang solid.

Sertifikat JavaScript dan jQuery
  • JavaScript: Variabel, fungsi, DOM, async/await.
  • jQuery: Manipulasi DOM, animasi, AJAX, plugin.
  • Interaktivitas: Integrasi fitur dinamis pada web.
  • Best Practices: Struktur kode, debugging, performa.

Sertifikat ini menegaskan penguasaan web dinamis dengan JS & jQuery.

Sertifikat SOLID
  • SRP: Kelas dengan satu tanggung jawab.
  • OCP: Terbuka untuk ekstensi, tertutup untuk modifikasi.
  • LSP: Subclass menggantikan superclass tanpa error.
  • ISP: Interface khusus untuk kebutuhan spesifik.
  • DIP: Ketergantungan pada abstraksi, bukan detail.

Sertifikat ini menegaskan penguasaan prinsip desain perangkat lunak.

Sertifikat MNCTA
  • Analisis Ancaman: Deteksi dan penilaian kerentanan.
  • Pertahanan Jaringan: Firewall, IDS, IPS.
  • Respon Insiden: Strategi pemulihan dan mitigasi.
  • Pemantauan: Audit keamanan dan pelaporan.
  • Kepatuhan: Penerapan GDPR, ISO 27001, dll.

Sertifikat ini menunjukkan kompetensi dalam cybersecurity dan analisis jaringan.
